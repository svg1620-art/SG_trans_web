# SG_Транскрибация — Веб-интерфейс

## Переменные окружения на Railway

| Переменная | Описание |
|---|---|
| `OPENAI_API_KEY` | Ключ OpenAI |
| `SECRET_KEY` | Любая случайная строка для сессий |
| `USERS` | JSON со списком пользователей |

## Формат USERS

```json
{"admin": "пароль1", "user2": "пароль2"}
```

По умолчанию: `{"admin": "admin123"}` — обязательно измени!

## Деплой на Railway

1. Залей папку в отдельный GitHub репозиторий
2. Railway → New Project → GitHub Repository
3. Добавь переменные окружения
4. Railway автоматически выдаст публичный URL
